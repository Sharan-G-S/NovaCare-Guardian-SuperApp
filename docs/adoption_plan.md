Adoption Plan
=============

Pilot Phase
-----------
- deploy in one district operations center
- integrate with existing cold chain reporting channel
- run shadow mode for two weeks before active decision support

Scale Phase
-----------
- onboard additional clinics in waves
- add localized policy thresholds
- build simple operator playbooks and escalation guides

Governance and Trust
--------------------
- retain full incident trace for auditability
- define operator override controls
- monitor false positives and missed incidents continuously

Impact Tracking
---------------
- incident response time
- spoiled dose count
- maintenance compliance rate
- clinic-level risk trend over time
